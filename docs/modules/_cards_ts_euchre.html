Module @cards-ts/euchre

@cards-ts/euchre

This is an implementation of the game Euchre in Typescript written on the @cards-ts framework. Euchre is a trick-taking game.

Running a card game with npx

npx @cards-ts/euchre

Running a card game locally

npm run start

Running the tests

npm test
