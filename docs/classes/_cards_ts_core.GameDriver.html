Class GameDriver<Handlers, State, Controllers, GameState, ResponseMessage, EventHandler>

Class that steps though the game using the game state transitions

Shouldn't be manually constructed - game factory should handle

Type parameters

  • Handlers: {} & SystemHandlerParams

    the custom event handlers that this game has

  • State: typeof STANDARD_STATES

    the state enum for this game

  • Controllers: IndexedControllers & { completed: CompletedController; state: GameStateController<State>; waiting: WaitingController }

    the state controllers for this game

  • GameState: GenericGameState<Controllers>

  • ResponseMessage: Message

    the response messages this game expects

  • EventHandler: EventHandlerInterface<Controllers, ResponseMessage>
